# Bundled ZIM Files

Optionally place `.zim` files in this folder to bundle them into the add-on image.

Examples:

- `kiwix/zims/wikipedia_pt_all_mini_2026-01.zim`
- `kiwix/zims/wikipedia_en_all_mini_2026-01.zim`

Notes:

- Large ZIM files make builds slow and images very large.
- For daily usage, prefer `/share/kiwix` or `/media/kiwix`.
