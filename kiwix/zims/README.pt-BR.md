# ZIMs Embutidos

Opcionalmente coloque arquivos `.zim` nesta pasta para embutir no build da imagem do add-on.

Exemplos:

- `kiwix/zims/wikipedia_pt_all_mini_2026-01.zim`
- `kiwix/zims/wikipedia_en_all_mini_2026-01.zim`

Notas:

- Arquivos ZIM grandes deixam o build lento e a imagem muito pesada.
- Para uso diario, prefira `/share/kiwix` ou `/media/kiwix`.
